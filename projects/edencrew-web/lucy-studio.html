  • Provider Instance — shared data providers across pages
  • Data Set — custom static or mock datasets for prototyping
  • Drag-and-drop field mapping from response to widget properties

Console & Debug

Full visibility inside
the editor

Lucy Studio's built-in console surfaces Studio logs and App logs side by side. Trace data binding calls, inspect widget state changes, and catch runtime errors without leaving your workspace.

  • Studio Log — editor-level events, build steps, deploy status
  • App Log — runtime Flutter logs from connected device or emulator
  • Filter by log level: Info, Warning, Error, Success
  • Click any log entry to jump to the originating widget
