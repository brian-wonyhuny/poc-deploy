01 — Overview

사이트 목적 & 타깃 유저

Edencrew 공식 홈페이지는 "AI 기반 솔루션 스타트업"의 신뢰도를 시각화하고, 핵심 제품인 Lucy Studio의 조기 채택자(얼리어답터)를 확보하기 위한 영업·마케팅 채널이다. 단순 소개 페이지를 넘어, 방문자가 제품을 이해하고 베타 신청·데모 예약까지 완료하는 전환 파이프라인으로 설계한다.

Primary Target
스타트업 창업자 & 앱 개발 담당자
코딩 없이 Flutter 앱을 빠르게 만들고 싶은 비개발자 또는 소규모 팀. 기존 FlutterFlow 사용자도 포함.
Secondary Target
에이전시 & 프리랜서 개발자
클라이언트 납품 속도를 높이려는 에이전시. Edencrew Platform의 화이트라벨·팀 플랜 관심군.
Tertiary Target
투자자 & 파트너사
Edencrew의 기술력·제품 로드맵·팀을 검토하는 VC·엔젤·파트너십 대상.
전환 목표 (Conversion Goals)
목표지표성공 기준 (론칭 후 90일)
베타 신청신청 폼 제출 수월 200+ 건
데모 예약Calendly 예약 완료월 30+ 건
제품 이해Lucy Studio 페이지 평균 체류2분 30초 이상
브랜드 신뢰팀·About 페이지 방문율전체 방문의 25%+
투자자 관심Contact 직접 문의월 5+ 건

03 — Page Requirements

페이지별 기능 요구사항

Platform — /platform
플랫폼 생태계 다이어그램
Lucy Studio를 중심으로 향후 제품·서비스가 연결되는 인터랙티브 맵. 각 노드 hover 시 툴팁으로 제품 설명 표시.
API 개요 섹션
외부 개발자가 Lucy Studio를 임베드·확장할 수 있는 API 존재를 알림. Waitlist CTA로 연결.
엔터프라이즈 티어 소개
팀 사이즈·권한 관리·SSO·전용 지원의 존재를 명시. 가격 미표기, "문의하기" CTA로 처리.
파트너 프로그램 배너
에이전시·리셀러·교육 기관 대상 파트너십 프로그램 예고. 이메일 신청 폼 또는 Notion 폼 임베드.
Pricing — /pricing
월간/연간 토글
토글 전환 시 가격 숫자 애니메이션. 연간 선택 시 절감 금액 배지 표시.
추천 플랜 강조
Pro 플랜 카드에 "Most Popular" 배지 및 accent 보더. 나머지 카드는 동일 높이로 정렬.
기능 비교 테이블
sticky 컬럼 헤더. 체크/크로스 아이콘 + 일부 셀은 수치값. 모바일에서 horizontal scroll 처리.
FAQ 아코디언
질문 클릭 시 답변 펼침/닫힘. 결제 수단·환불 정책·플랜 업그레이드 절차 포함.
베타 기간 안내 배너
현재 베타 기간임을 명시 + 무료 신청 CTA. 공식 출시 후 제거 가능한 컴포넌트로 분리.
Resources — /resources
콘텐츠 카테고리 필터
블로그 / 릴리스 노트 / 튜토리얼 탭 필터. 초기에는 "Coming Soon" 슬레이트로 표시.
뉴스레터 구독 폼
이메일 입력 + 구독 버튼. 백엔드는 Resend 또는 Notion DB 연동.
템플릿 갤러리 그리드
앱 스크린샷 썸네일 + 카테고리 태그 + "Preview" CTA. MVP에서는 3–6개 샘플 템플릿.
커뮤니티 링크 패널
Discord 초대 링크 / YouTube 채널 / GitHub(공개 레포 있을 경우) 아이콘 카드.

05 — CTA Strategy

CTA 전략 — 페이지별 행동 유도 흐름

방문자가 Home → Lucy Studio → Pricing → 베타 신청의 깔때기를 자연스럽게 통과하도록 각 페이지의 CTA를 설계한다.

1
Home
베타 신청하기 (Primary) + Lucy Studio 살펴보기 (Secondary)
첫 방문자가 제품을 모르는 상태. Primary는 즉시 전환 유도, Secondary는 제품 이해 흐름으로 진입.
2
Lucy Studio
무료로 시작하기 (Primary) + 데모 영상 보기 (Secondary)
각 피처 섹션 끝마다 인라인 CTA 반복 배치. 페이지 하단 풀-블리드 배너에서 최종 전환.
3
Platform
파트너십 문의하기 (Primary) + API Waitlist 신청 (Secondary)
B2B·개발자 타깃. 즉시 결제보다 "관계 시작" 행동 유도.
4
Pricing
플랜 시작하기 / 무료 시작 (Primary) + 엔터프라이즈 문의 (Secondary)
결정 단계 방문자. 플랜 카드별 개별 CTA 배치. 베타 기간 중 "무료"를 전면에.
5
Resources
뉴스레터 구독 (Primary) + Discord 참여 (Secondary)
당장 전환하지 않는 탐색형 방문자를 커뮤니티로 유입.
✓
전환 완료
베타 신청 완료 → 확인 이메일 발송
신청 완료 페이지에서 Discord 참여 + 소셜 공유 유도.

반응형 요구사항

3개 브레이크포인트 기준. Next.js + Tailwind의 sm / md / lg 프리픽스로 구현.

375px +
  • GNB 햄버거 메뉴
  • 카드 그리드 1컬럼
  • 비교 테이블 가로 스크롤
  • CTA 버튼 full-width

768px +
  • GNB 링크 표시
  • 카드 그리드 2컬럼
  • 푸터 2컬럼
  • Pricing 플랜 카드 2×2
Desktop
1280px +
  • GNB 전체 링크 + CTA
  • 카드 그리드 3–4컬럼
  • 푸터 4컬럼
  • max-width: 1440px 중앙 정렬
항목요구사항
터치 타깃최소 44×44px (WCAG 2.5.5 기준)
이미지Next.js <Image> 컴포넌트, WebP + srcset 자동 생성
폰트 로딩next/font로 Montserrat·Inter 자체 호스팅, FOUT 방지
애니메이션prefers-reduced-motion 미디어 쿼리 존중
컨트라스트본문 최소 4.5:1 / 대형 텍스트 3:1 (WCAG AA)
07 — MVP Scope

MVP 1차 론칭 범위

조기 트랙션 확보를 위해 베타 신청 파이프라인 완성에 집중한다.

MVP 포함
  • Home 페이지 (완성)
  • Lucy Studio 페이지 (완성)
  • Platform 페이지 (기본 5섹션)
  • Pricing 페이지 (플랜 카드 + FAQ)
  • Resources 페이지 (뉴스레터 + 커뮤니티)
  • GNB + Footer 공통 컴포넌트
  • 베타 신청 폼 + /beta/thanks 페이지
  • 반응형 (모바일·태블릿·데스크톱)
  • 기본 SEO (title/description/OG 태그)
  • Vercel 배포 + 커스텀 도메인
  • Google Analytics 4 연동
2차 이후
  • 블로그 / 릴리스 노트 CMS
  • 템플릿 갤러리 (실 데이터)
  • API 문서 페이지
  • 다크/라이트 테마 토글
  • i18n (한/영 전환)
  • Enterprise 견적 자동화
  • 인앱 데모 임베드
  • Changelog 자동 생성
  • 채용 페이지
