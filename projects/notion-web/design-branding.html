Page Architecture
Homepage Section Blueprint
홈페이지 섹션 순서와 각 섹션의 역할. 총 9개 섹션으로 구성됩니다.
1
Hero — Lucy Platform 카테고리 + Edencrew 신뢰
메인 카피 + 서브 설명 + CTA 2개 (Download Studio / Contact Sales). 다크 네이비 배경.
2
Proof Strip — 핵심 수치 4개
40%↓ 제작기간 / 60%↓ 비용 / 10+ 연 경험 / 12개 클라이언트. 라이트 배경, 숫자 강조.
3
Problem — 같은 화면을 3번 만드는 비효율
MTS/HTS/WTS를 따로 개발하는 고객 Pain Point 시각화.
4
Solution — 통합 워크플로우
Figma → Lucy Studio → SDK → TeamCloud → Deploy → LAMP 플로우 다이어그램.
5
Product Suite — 5개 제품 라인업
Lucy Studio / Lucy SDK / TeamCloud / Lucy Deploy / LAMP 각각 카드로 소개.
6
Technology Trust — 기술 신뢰성
Flutter/Impeller Runtime, OTA 업데이트, SDUI, JSON UI, FIDO/E2E 보안.
7
References — 클라이언트 · 산업 · 국가
국내 9개 + 글로벌 3개 증권사 로고. 승인 받은 범위 내에서만 공개.
8
Company — Thesis + 연혁
"이든크루는, 프로젝트 경험을 플랫폼 자산으로 전환합니다." + 2015~2026 타임라인.
9
CTA — 도입 문의 · 소개서 · 기술 미팅
세 가지 전환 경로로 분기. Contact Sales / 소개서 PDF 요청 / 기술 미팅 예약.
